# Ocean-View-Resort - Reservation System
By: Rushdi Ramzaan

## Project Overview:
This project uses Java EE technologies to implement a distributed web-based room reservation system for Ocean View Resort. The system uses design patterns, appropriate testing techniques, and SOLID principles in its three-tier architecture.

## Technologies Used
1. JavaEE
2. Tomcat
3. Maven
4. JDBC
5. JUnit

## Architecture Overview
This system follows a layered architecture:

Presentation Layer → Controller → Service → Repository → Database

## Features
- User Authentication
- Add Reservation
- View Reservation
- Calculate Bill
- Reporting
- Testing and Automation
